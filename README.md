# 乐健一键跑步、签到app
仅支持Android， 安装后一键签到、刷跑步、课程签到
支持自选跑步方式，可以自选跑步位置（目前支持了电子科大两校区和四川农业大学）
支持自选活动一键报名并签到
支持课程一键签到
在[release](https://github.com/Foreverddb/FuckLegym/releases/)中下载需要的app版本即可
